2FA
Two Factor Authentication (see MFA)
ACL
Access Control List
AD
Active Directory (see LDAP)
ALE
Annualized loss expectancy (ALE): The total loss in dollars per year due to a specific incident. The incident might happen once or more than once; either way, this number is the total loss in dollars for that particular type of incident. It is computed with the following calculation:
SLE × ARO = ALE
AMAC
NCPS Advanced Malware Analysis Center, an organization within CISA.
ARO
Annualized Rate of occurrence.  The number of times per year that the specific incident occurs.
ARP
Address Resolution Protocol
APA
America Psychological Association.  They are famous for the APA style guide.  There is also a bibliographic reference generator tool.
APT
Advanced Persistent Threat
AS
Authentication Server (Kerberos)
AUP
Acceptable Use Policies
BPA
Business Partners Agreement - a type of contract that can establish the profits each partner will get, what responsibilities each partner will have, and exit strategies for partners.
Bot
Robot or a robot network
CAC
? (from https://public.cyber.mil/announcement/stig-update-disa-has-released-the-canonical-ubuntu-20-04-lts-stig/ and elsewhere within DISA)
CAM
Content Addressable Memory
CSAM
cyber security asset and management
CCI
Control Correlation Identifier.  The most recent of these is from February 2017.  If it is that old, is it still useful?
CCTV
Closed Circuit Television
CERT
Computer emergency response team (Not to be confused with US-Computer Emergency Readiness Team.)
CDM
Continuous Diagnostics and Mitigation
CIA
Confidentiality, integrity, availability (from https://login.codingdojo.com/m/285/9008/61222).  See Data Integrity: Identifying and Protecting Assets Against Ransomware and Other Destructive Events  [2].
CIP
Critical Infrastructure Protection
CIS
Center for Internet Security.  See the 18 CIS critical security controls.  Check my logbook November 8th: how am I doing?
CISA
Cybersecurity and Infrastructure Security Agency.
CompTIA
Computing Technology Industry Association (See the about us page.)
COBIT
An ISACA framework.  I don't understand the graphic about COBIT.  It talks about the ITIL®4 key components.  There are the Guiding Principles, governance, service value chain, continual improvement, and practices.
CS
Career Services? (First encountered on the career services roadmap).
CSM
Career Services Manager?
CySA+
CompTIA Cybersecurity Analyst.
DAC
Discretionary Access Controls (see Mandatory Access Controls)

 Dynamic Application Security Testing (from a talk by Dawn E. Collett, Free Security for Open-Source Projects)

DISA
Defense Information Systems Agency

Data Loss Prevention (see Information Loss Prevention, a synonym)
